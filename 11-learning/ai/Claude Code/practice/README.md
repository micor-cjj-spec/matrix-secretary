# Claude Code Practice

> 本目录用于 Claude Code 学习过程中的小练习、作业提交、批阅记录和阶段评分。

## 学习方式

你不用一开始直接看源码。学习过程采用下面的节奏：

```text
听讲解 → 做小练习 → 本地提交 → 我批阅 → 打分 → 讲解薄弱点 → 进入下一课
```

## 目录规划

```text
practice/
├── README.md
├── exercise-01-agent-loop.md
├── exercise-02-tool-calling.md
├── exercise-03-context-memory.md
├── exercise-04-permission-safety.md
├── submission-template.md
└── grading-rubric.md
```

## 文件说明

| 文件 | 用途 |
|---|---|
| `exercise-01-agent-loop.md` | 第一课练习：理解 Agent Loop。 |
| `exercise-02-tool-calling.md` | 第二课练习：理解工具调用。 |
| `exercise-03-context-memory.md` | 第三课练习：理解上下文与记忆。 |
| `exercise-04-permission-safety.md` | 第四课练习：理解权限与安全。 |
| `submission-template.md` | 你的作业提交模板。 |
| `grading-rubric.md` | 批阅和打分标准。 |

## 提交流程

1. 在本地完成对应练习文件。
2. 按 `submission-template.md` 的格式写答案。
3. 提交到 GitHub。
4. 告诉我提交记录或让我查看最新变更。
5. 我会从理解准确性、表达清晰度、结构完整性、迁移能力四个维度批阅并打分。
